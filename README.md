Proyecto DIVIPOLA multilista

Este proyecto lee un archivo CSV de DIVIPOLA y construye una multilista con la jerarquia Colombia, Departamentos y Municipios.

Archivos principales

node.py
country.py
department.py
city.py
doublelinklist.py
files.py
main.py
templates/index.html

Ejecucion

pip install -r requirements.txt
python main.py

La aplicacion corre en el puerto 8080.
